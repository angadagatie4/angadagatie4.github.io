Les balises

Voici quelques-unes des balises htlm les plus utilisées :

  • - < h1> pour les titres originaux
  • - < p> pour les paragraphes
  • - < a> pour les liens
  • - < img> pour les images
  • < ul> et < li> -pour les listes non ordonnées
